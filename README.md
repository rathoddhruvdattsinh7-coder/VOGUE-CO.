# 🛍️ Vogue & Co. - Fashion E-Commerce Website

A modern and responsive fashion e-commerce website built using **HTML, CSS, and JavaScript**. This project showcases a clean shopping experience with product browsing, filtering, search functionality, wishlist, shopping cart, dark mode, and a premium UI.

> 🚧 **Project Status:** Work in Progress  
> This project is actively being developed, and new features are added regularly.

---

## ✨ Features

- 🛒 Shopping Cart
- ❤️ Wishlist
- 🔍 Product Search
- 🎯 Advanced Product Filters
- 🌙 Dark Mode
- 📱 Fully Responsive Design
- ⭐ Product Ratings
- 🏷️ Category-wise Products
- 📦 Recently Viewed Products
- 🔥 Modern & Interactive UI
